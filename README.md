## Description

Back-end development for Dungeons & Dragons character tracking using Node.js, Express & MongoDB. <br>
Front-end development using Pug Templates & CSS styling. <br>
Password encryption using bcryptjs, and backend-frontend communication through axios.

## Usage Information

Dungeons &amp; Dragon character creation &amp; viewing API<br>
Based on the natours project previously posted, remaking the Models & functionality. <br>
Some of the features (e.g. updating the inventory) only work through the backend, like MongoDBCompass, as this was made for personal use. <br>
Live Preview: https://dnd-nick.herokuapp.com/
